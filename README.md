
# Shoot'em Up

I'm trying to make a simple Shoot'em Up in a functional programing way.

To test it :

    git clone https://github.com/bcolucci/shoot-em-up.git \
      && cd shoot-em-up \
      && npm i \
      && bower i
      && npm start

    google-chrome http://localhost:3000/

## Commands

- **Escape** Pause
- **Space** Fire
- **Left/Right Arrow** Go left/right
