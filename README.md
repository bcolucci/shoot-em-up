
# Space Invader

I'm trying to make a simple Space Invader in a functional programing way.

To test it :

    git clone https://github.com/bcolucci/spaceinvader.git \
      && cd spaceinvader \
      && bower i
      && google-chrome index.html

## Commands

- **Escape** Pause
- **Space** Fire
- **Left/Right Arrow** Go left/right
